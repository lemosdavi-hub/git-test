# Git Test

Projeto de testes para aprendizado de Git, GitHub e ambientes virtuais em Python.

## Ambiente

Crie e ative um ambiente virtual e instale as dependências:

```bash
pip install -r requirements.txt
```

## Estrutura do projeto

```text
teste-git/
├── src/
├── notebooks/
├── data/
├── tests/
├── requirements.txt
├── README.md
├── .gitignore
└── .venv/
```
